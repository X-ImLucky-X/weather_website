# 🌦️ Weather Website

A responsive weather application built with **HTML, CSS, and JavaScript** that provides real-time weather information for cities around the world. Simply search for a city to view its current weather conditions, temperature, humidity, and more.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- 💧 Humidity information
- ☁️ Weather condition and description
- 📍 Displays city/location details
- 📅 Shows the current date
- ⚠️ Handles invalid city names gracefully
- 📱 Responsive and user-friendly interface

---

## 🛠️ Tech Stack

- **HTML5** – Structure and content
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6)** – API integration and application logic
- **Google Fonts** – Custom typography
- **Google Material Icons** – Icons and UI elements
- **Weather API** – Fetches real-time weather data

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/X-ImLucky-X/weather_website.git
```

### 2. Navigate to the project directory

```bash
cd weather-website
```

### 3. Configure the API Key

Obtain an API key from your preferred weather service (such as OpenWeatherMap) and add it to the JavaScript file where indicated.

```javascript
const API_KEY = "YOUR_API_KEY";
```

### 4. Run the project

Open **index.html** in your preferred web browser.

No additional installation is required.

---

## 🎮 Usage

1. Launch the application.
2. Enter the name of any city.
3. Click the **Search** button.
4. View the current weather details including:
   - Temperature
   - Humidity
   - Weather condition
   - Location
   - Current date

## 🌐 API

This project uses a weather API to retrieve live weather data.

You will need your own API key before running the application.

Example response includes:

- Temperature
- Humidity
- Weather description
- Wind speed
- Location information
